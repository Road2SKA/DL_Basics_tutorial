# DL_Basics_tutorial
Hands-on for the DL Basics lectures

Tutorial 1: [classification_and_interpretability.ipynb](https://github.com/Road2SKA/DL_Basics_tutorial/blob/main/classification_and_interpretability.ipynb). Built upon Michelle Lochner's [deep learning tutorial](https://github.com/MichelleLochner/ml-tutorials/blob/main/tutorial-deep-learning.ipynb). Adds a custom CNN implementation to play with, and an application of an interpretability tool.

Tutorial 2: [this_galaxy_does_not_exist.ipynb](https://github.com/Road2SKA/DL_Basics_tutorial/blob/main/this_galaxy_does_not_exist.ipynb). Implements a very simple diffusion model to generate fake galaxies.

Both tutorials make use of the GalaxyZoo dataset from [here](https://github.com/MichelleLochner/ml-tutorials/tree/main/data)
